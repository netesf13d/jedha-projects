## Deployment

### Local deployment

```console
$ export $API_URL="<url_of_the_API_server>"
$ streamlit run app.py --server.address localhost --server.port 5000
```

```powershell
$ $env:API_URL="<url_of_the_API_server>"
$ streamlit run app.py --server.address localhost --server.port 5000
```
